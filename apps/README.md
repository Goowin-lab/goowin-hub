# apps

Contiene las aplicaciones ejecutables del producto.

## Carpetas

- `api`: backend NestJS. Expondra la API, contendra los casos de aplicacion, integraciones y acceso a datos via Prisma.
- `mobile`: aplicacion React Native. Contendra la experiencia mobile de Goowin Hub para iOS y Android.

Cada aplicacion debe poder evolucionar con sus propios scripts, dependencias, pruebas y configuracion.
