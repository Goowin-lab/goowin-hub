# apps/api

Backend de Goowin Hub basado en NestJS.

## Proposito

Esta carpeta alojara la API del SaaS, la configuracion de NestJS, Prisma, validaciones, autenticacion, integraciones externas y pruebas del backend.

## Estado actual

Solo existe la estructura base. No hay modulos NestJS, controladores, servicios, entidades, endpoints ni logica de negocio.
