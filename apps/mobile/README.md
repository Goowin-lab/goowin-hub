# apps/mobile

Aplicacion mobile de Goowin Hub basada en React Native.

## Proposito

Esta carpeta alojara la experiencia mobile para iOS y Android, incluyendo navegacion, vistas, componentes, estado de cliente y comunicacion con la API.

## Estado actual

Solo existe la estructura base. No hay pantallas, navegacion, componentes visuales ni logica de producto.
