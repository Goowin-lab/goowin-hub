# Business Rules

## Proposito

Este archivo registrara reglas de negocio antes de convertirlas en codigo.

## Estado actual

No hay reglas de negocio definidas.

## Criterio de uso

Antes de implementar una funcionalidad, se debe documentar:

- Regla o politica del negocio.
- Actores involucrados.
- Casos validos.
- Casos invalidos.
- Efectos sobre datos.
- Impacto en API y mobile.
