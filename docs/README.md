# docs

Documentacion tecnica y de producto de Goowin Hub.

## Archivos

- `ARCHITECTURE.md`: decisiones tecnicas, limites de sistema y evolucion prevista.
- `DATABASE.md`: decisiones de persistencia, PostgreSQL y Prisma.
- `BUSINESS_RULES.md`: reglas de negocio antes de implementarlas en codigo.
- `ROADMAP.md`: fases del proyecto y prioridades.
- `API_SPEC.md`: contrato futuro de la API.

La documentacion debe actualizarse antes o durante cualquier cambio relevante de arquitectura.
